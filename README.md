## 功能介绍

🎉 支持小程序中如下操作：

- [x] 绘制矩形
- [x] 绘制圆形
- [x] 绘制图片
- [x] 绘制文字
- [x] 导出绘制

## 相关方法

```js
// 绘制矩形或图片
drawRect(canvas, ctx, x, y, width, height, radius, bgColor, path, isClip)
```

```js
// 绘制文字
drawText(width, ctx, text, top, size, color = "#000")
```

```js
// 导出绘制
exportData()
```
